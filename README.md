# Automatic Airport Split Flap 
Airport Solari board. To run on your machine simply copy the repository or download the 3 files: index.html docker-compose.yml and Dockerfile into the same folder. Then run " docker compose up -d --build " now your airport dashboard is running on your machine at port 8055 http://localhost:8055 

This app was a simple iteration of html and made mobile friendly, ready to use on fullscreen either on laptops, tablets or mobile. 

Why?
I find it relaxing, probably others will too.
